## **TROUBLESHOOTING**

### **Problem: "npm not found" when trying CLI**
**Solution:** Use drag-and-drop method instead (Method 3A)

### **Problem: "Files won't upload"**
**Solution:** 
- Check file size (under 100MB should work)
- Try zipping folder first
- Use GitHub method instead

### **Problem: "Site loads but looks broken"**
**Solution:**
- Check browser console for errors (F12)
- Verify all files uploaded correctly
- Check file paths are relative (not absolute)

### **Problem: "Can't see the site after deploy"**
**Solution:**
- Wait 2-3 minutes for DNS to propagate
- Clear browser cache (Ctrl+Shift+R)
- Try incognito/private window
